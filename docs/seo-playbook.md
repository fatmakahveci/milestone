# SEO Playbook

Milestone is primarily a research workflow and Streamlit demo, so SEO impact is naturally limited by the app architecture. The highest-value discoverability work is therefore documentation-led.

## Target Queries

- bacterial wgMLST workflow
- wgMLST strain typing pipeline
- bacterial strain typing web app
- PubMLST benchmark workflow
- cgMLST wgMLST schema QC

## Applied in this repository

- README title and opening copy now use `wgMLST`, `bacterial strain typing`, `benchmark`, and `schema QC` phrasing directly.
- The Streamlit page title is now `Milestone wgMLST demo | bacterial strain typing`.
- Public importer, benchmark, matrix, and validation-bundle sections are described with searchable headings.
- Comparative-analysis and benchmark-source docs now create internal links around core scientific terms.
