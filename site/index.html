Milestone

Milestone is an open wgMLST workflow for bacterial strain typing, schema creation, allele calling, benchmark validation, public schema import, and browser-based demo execution.

wgMLST workflow

Run schema creation, allele calling, matrix comparison, batch comparison, and truth-set benchmarking from one CLI.

Public data import

Import PubMLST and EnteroBase assets, then freeze selected isolates into provenance-aware benchmark packs.

Validation

Use schema QC, benchmark reports, validation bundles, and tool-benchmark harnesses to support reproducible academic evaluation.
