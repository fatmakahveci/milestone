Milestone

Milestone is a local wgMLST workflow for bacterial allele calling, profile comparison, schema QC, public schema import, and validation support.

View Repository How To Use Milestone

One workflow

Run schema creation or import, allele calling, profile comparison, schema QC, and benchmarking in one local project.

Public schema import

Import PubMLST and EnteroBase assets, then freeze selected isolates into provenance-aware benchmark packs.

Validation support

Use schema QC, benchmark reports, validation bundles, and reporting outputs to keep analyses easier to rerun and inspect.
