Milestone

Milestone is a Streamlit web app and CLI workflow for wgMLST: public schema import, allele calling, profile comparison, matrix/batch analysis, benchmark validation, and publication-ready packaging.

View Repository Quick Start How To Use Milestone Benchmark Page

Important: This GitHub Pages site is a static overview. The interactive Milestone UI runs with streamlit run webapp/app.py or via your own Streamlit deployment.

Web UI + live jobs

Run demo or real jobs from the browser with job-state tracking, live log streaming, and downloadable run outputs.

Public schema import

Import PubMLST and EnteroBase assets, filter cgMLST/wgMLST schemes, and freeze isolate-backed benchmark packs.

Scientific validation

Use schema QC, per-locus benchmarking, novel allele export, and publication-readiness packaging for reproducible outputs.
