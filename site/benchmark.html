Benchmark and Validation

Milestone supports truth-set benchmarking, provenance-aware species packs, frozen public snapshots, per-locus mismatch summaries, and validation bundles suitable for reproducible bioinformatics reporting.

  • Species benchmark packs for Escherichia coli, Neisseria meningitidis, and Acinetobacter baumannii
  • PubMLST-backed benchmark-pack import and freezing
  • HTML, TSV, and JSON benchmark reports
  • Novel-allele export and per-locus discordance visibility
  • Tool harness scaffolding for Milestone, chewBBACA, and MentaLiST

Run locally: streamlit run webapp/app.py for interactive benchmark run control and downloadable outputs.
